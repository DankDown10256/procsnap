procsnap is a process profiler inspired by the suckless philosophy

usage:

    procsnap <pid>
    procsnap --json <pid>
    procsnap --diff <pid>
    procsnap -g <pid>
    
you can open issues I'm open to all feedbacks and help.

Thanks.
